## ğŸ¬ Build Process

1. You edit `app/page.js`
2. Save the file
3. Next.js sees the change
4. Page auto-refreshes (in dev mode)
5. When ready, you run `npm run build`
6. Next.js creates optimized files in `.next/`
7. You deploy to Vercel
8. Vercel serves the optimized files
9. Users see your beautiful proposal!
